Responsive Design: Canada Council for the Arts

Building a meaningful user interface from a clear and deliberate brand voice for a government agency website.

TIMELINE

5 weeks

ROLE

Developed a brand style guide and applied UI elements to responsive mock-ups.

DELIVERABLES

UI style guide, brand voice, information architecture/site map, sketches & wireframes, responsive high-fidelity mock-ups

TOOLS

Figma, Miro.

Current Website

Website Analysis / Card Sorting

esearch: Website Annotation and Card Sorting helped identify the problems and potential solutions.

  • WEBSITE ASSESSMENT
  • Testing Users identified few issues to improve on:
  • 1.Some links were hard to locate.
  • 2.Too many links on one page which lead to same page.
  • 3.Overall website lacks contrast.

REBRAND: Improving Information Architecture of the website and Brand Voice.

SOLUTION

The current website of the Canada Council for the Arts was cluttered with too much information. The overall website lacked contrast making it hard for the users to find a necessary information. To solve the problem, new site map was created to improve information architecture of the website. New brand identity was created to make the visual design bold and engaging.
